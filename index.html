Projects

Analysis of machine learning techniques for detecting credit card fraud

Empirically evaluated several techniques for credit card fraud detection on a Kaggle-hosted dataset sourced from the collaborative effort of Worldline and the Machine Learning Group (MLG) of ULB (http://mlg.ulb.ac.be).

To that end, this project sought to evaluate the impact of SMOTE (a relatively popular data-level method for dealing with unbalanced datasets) on classification performance, along with another technique that has seen some use which combines SMOTE and TomekLinks (which performs undersampling rather than oversampling, as with SMOTE).

In combination with the above data-level methods, this project sought to assess and evaluate the binary classification performance of various machine learning techniques. A mix of classical machine learning were selected in concordance with those evaluated by the reviewed literatures, as well as a few ensemble techniques for comparison. These were: Logistic Regression, Naïve Bayes, Random Forest and a Bagging ensemble with decision trees. An additional experiment using Cost-Sensitive Logistic Regression and Random Forest was done to evaluate the performance of data-level versus algorithm-level methods for learning on imbalanced datasets
